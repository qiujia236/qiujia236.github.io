用 hugo 搭建个人博客

December 30, 2019

如何使用 hugo 搭建个人博客

  1. 下载 hugo，到https://github.com/gohugoio/hugo/releases 下载 hugo_0.62.0_Windows-64bit.zip，解压，将 hugo.exe 放到个人目录里面，并且设置环境变量，path 为你的个人目录路径,安装完成后使用 hugo version 检测 hugo 是否安装成功，如果出现 Hugo Static Site Generator v0.62.0-6608F155 windows/amd64 BuildDate: 2019-12-23T10:26:18Z 的字样，则表明安装成功；

  2. 使用 hugo new site xxx 创建一个 hugo 站点，其中 xxx 为站点的名字；

  3. 添加主题，使用 cmder 软件输入 cd xxx 命令进入站点，在站点里面输入 git init 和 git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 两条命令获取主题，最后输入 echo ‘theme = “ananke”’ » config.toml 添加主题，在 config.toml 中可以修改相关配置，例如你的标题，你的语言编码，还有 URL，其中 URL 必须为你的域名，比如 xxx.github.io 域名； !(/images/toml.png)

  4. 添加内容，使用 hugo new posts/xxx.md 命令添加博客内容，xxx 为要编辑博客文件的名字，根据个人喜好取名。 !(/images/md文件标题头.png)

    .md 文件中的标题，date 和 draft 均可以根据需要修改。draft 修改为 false，在 hugo 启动后便可以查看整体效果；

  5. 启动 hugo，受用 hugo server -D 命令启动，在浏览器中输入 http://localhost:1313/ 便可以访问博客； !(/images/本地博客.png)

  6. 使用 hugo -D 会在./public/生成博客的静态页面，上传到 github 后，可以通过 xxx.github.io 观看。 !(/images/github博客.png)
